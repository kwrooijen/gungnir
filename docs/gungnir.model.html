register!

(register! model-map)

Adds the model-map to the current available models for Gungnir. You can add multiple models at once, or add new ones over time.

The following format is accepted. Keys are the name of model, and the value should be a Malli :map

(gungnir.model/register!
 {:user [:map ,,,]
  :post [:map ,,,]
  :comment [:map ,,,]})
